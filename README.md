# Simulación de Máquina de Galton
Consiste en una simulación física y estadística de la **Máquina de Galton**. Obtiene desiciones aleatorias individuales ($+1$ hacia la derecha o $-1$ hacia la izquierda)
## Características del programa
* **Simulación:** Modela el comportamiento de **3000 canicas** cayendo de forma individual.
* **Obstáculos:** Cuenta con **12 niveles** de obstáculos (desiciones aleatorias).
* **Análisis visual:** Genera un historigrama automatizado utilizando la librería `matplotlib`.
* **Presición de los datos:** Muestra la cantidad exacta de las canicas reecolectadas encima de cada contenedor.
## 🛠️ Requisitos e Instalación

El programa requiere de Phyton y la biblioteca externa `matplotlib` para el despliegue gráfico.
1. Asegurate de tener isntalado Phyton en tu sistema.
2. Instala la librería necesaria ejecutando el siguiente comando en tu terminal (**CMD** o **PowerShell**):

pip install matplotlib
